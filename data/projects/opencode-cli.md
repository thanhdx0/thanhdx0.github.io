Opencode is an AI-powered CLI editor designed specifically for software engineering tasks. It provides an intelligent, interactive approach to coding with the power of AI.

## Features

- **Interactive CLI Interface**: Navigate and edit files directly from your terminal
- **AI-Powered Assistance**: Leverages Minimax M2.5 model for intelligent coding help
- **Codebase Awareness**: Understands your project structure and context
- **Multi-step Task Execution**: Breaks down complex tasks into manageable steps
- **Shell Integration**: Execute commands directly within the editor
- **Safety First**: Built-in safety checks and conventions enforcement

## Technologies Used

The project is built with modern technologies including TypeScript and Node.js, ensuring type safety and maintainability.

## Getting Started

```bash
npm install -g opencode
opencode --model minimax
```

## Why Opencode?

Traditional AI coding assistants work well for single-prompt tasks, but Opencode excels at:
- Complex, multi-step refactoring tasks
- Understanding existing codebase patterns
- Maintaining code conventions automatically
- Interactive debugging sessions
